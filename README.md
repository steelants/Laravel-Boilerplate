## Content
- User Management
- Job Management
- Cache Management
- Log Viewer
- Audit

## Install

```bash
composer require steelants/laravel-boilerplate
composer install
php artisan boilerplate:install
```

## Development

### Creation of symlinks for dev environment:

```bash
ln -s ./package/boilerplate/stubs/resources/ resources
```
